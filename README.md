# Usage
<!-- usage -->
```sh-session
$ npm install -g mmctl
$ mmctl COMMAND
running command...
$ mmctl (-v|--version|version)
mmctl/0.0.0 darwin-x64 node-v15.5.1
$ mmctl --help [COMMAND]
USAGE
  $ mmctl COMMAND
...
```
